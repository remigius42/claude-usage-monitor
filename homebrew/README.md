# Homebrew Tap for Claude Usage Monitor

![License](https://img.shields.io/badge/license-MIT-blue.svg)

Homebrew formulae for [Claude Usage Monitor](https://github.com/remigius42/claude-usage-monitor).

## Installation

```bash
brew tap remigius42/claude-usage-monitor
brew install claude-usage-monitor
```

## Available Formulae

| Formula                | Description                                                                                 |
|------------------------|---------------------------------------------------------------------------------------------|
| `claude-usage-monitor` | Monitor Claude API usage in your menu bar (SwiftBar/Polybar) or Claude Code CLI status line |

## Platform Support

- **macOS**: Installs SwiftBar plugin
- **Linux**: Installs Polybar module
